linconnect-windows
==================

Mirror Android notifications on a windows.

LinConnect Client: https://github.com/hauckwill/linconnect-client/



Installation
------------

**Requirements**

* .Net Framework


**Configure**

* Run both server(this application) and client(android app)

* Make sure both are connected to same wifi 

* Add IPaddress and port number to custom IP address in the client
